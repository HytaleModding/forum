# Theme

![License](https://img.shields.io/badge/license-MIT-blue.svg) [![Latest Stable Version](https://img.shields.io/packagist/v/hytale-modding/theme.svg)](https://packagist.org/packages/hytale-modding/theme) [![Total Downloads](https://img.shields.io/packagist/dt/hytale-modding/theme.svg)](https://packagist.org/packages/hytale-modding/theme)

A [Flarum](https://flarum.org) extension. Hytale Modding&#39;s forum theme.

## Installation

Install with composer:

```sh
composer require hytale-modding/theme:"*"
```

## Updating

```sh
composer update hytale-modding/theme:"*"
php flarum migrate
php flarum cache:clear
```
