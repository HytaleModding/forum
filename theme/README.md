# Theme

![License](https://img.shields.io/badge/license-MIT-blue.svg) [![Latest Stable Version](https://img.shields.io/packagist/v/hytalemodding/theme.svg)](https://packagist.org/packages/hytalemodding/theme) [![Total Downloads](https://img.shields.io/packagist/dt/hytalemodding/theme.svg)](https://packagist.org/packages/hytalemodding/theme)

A [Flarum](https://flarum.org) extension. Hytale Modding&#39;s forum theme.

## Installation

Install with composer:

```sh
composer require hytalemodding/theme:"*"
```

## Updating

```sh
composer update hytalemodding/theme:"*"
php flarum migrate
php flarum cache:clear
```
